Featured Case: BT Mobile Apps (Great Britain)

Legacy Mobile Stack Redesign and Cost Reduction

Challenge

BT faced aging iOS and Android applications built by different vendors, creating high maintenance overhead, uneven UX, and slower release cycles for platform updates.

Solution

People More redesigned the app experience, collaborated with BT's backend team to define modern APIs, and implemented a single React Native codebase for both platforms.

Process

  1. UX/UI redesign aligned to updated Apple and Google standards.
  2. Joint API planning with backend stakeholders.
  3. Cross-platform application rebuild with shared architecture.
  4. Release preparation with long-term maintenance efficiency in mind.

Measured Results

  • Development cost lowered by almost half after migration to shared codebase.
  • Future maintenance costs reduced due to one core mobile architecture.
  • Faster adaptation path for iOS and Android platform changes.

Client Quote

"Peoplemore successfully delivered all project features within the scheduled timeline and budget constraints, while staying flexible during updates."
