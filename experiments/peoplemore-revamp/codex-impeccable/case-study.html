Featured case: BT mobile apps modernization

Source basis: scraped pages/case-study.md and scraped pages/homepage.md.

Client context

BT needed support for legacy iOS and Android applications that had been developed over time by different vendors. The product needed modern UX updates and sustainable development economics.

Challenge

  • Fragmented legacy code and uneven app consistency across platforms.
  • Need to support newer iOS and Android versions without recurring rework spikes.
  • Pressure to reduce delivery and maintenance cost while improving UX quality.

Solution

  • UX/UI redesign for a unified cross-platform experience.
  • Close collaboration with client backend teams to define updated APIs.
  • Single React Native code base compiling to native iOS and Android apps.

Process milestones

  1. Legacy assessment and release-risk mapping.
  2. Cross-platform UX/UI redesign and API alignment.
  3. React Native implementation and platform validation.
  4. OS version support updates and rollout stabilization.
Measured result

Development cost was lowered by almost half, and the unified technology stack reduced future maintenance overhead.

Country: Great Britain | Tech scope: React Native, UX/UI, Design, Objective-C, Java.
