# Project context

This is a GitHub Pages repository for `igor-g-pm.github.io`, served via Jekyll.

- Relative links in Markdown files resolve correctly on GitHub Pages (e.g. `experiments/peoplemore-revamp/README.md`)
- The `experiments/` directory holds standalone HTML/CSS/JS prototypes — they are not Jekyll-processed
- Main branch is `main`; pushes to `main` trigger automatic deployment
